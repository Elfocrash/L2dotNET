You'll need the following to upgrade clan level:
Level 1 - SP : 30,000, Adena : 650,000
Level 2 - SP : 150,000, Adena : 2,500,000
Level 3 - SP : 500,000, Item : Blood Mark
Level 4 - SP : 1,400,000, Item : Alliance Manifesto
Level 5 - SP : 3,500,000, Item : Seal of Aspiration
Level 6 - Clan Reputation : 10,000, Clan Members : More than 30
Level 7 - Clan Reputation : 20,000, Clan Members : More than 80
Level 8 - Clan Reputation : 40,000, Clan Members : More than 120
Get the Blood Mark by hunting the Blood Queen in Dragon Valley.
To get the Aliance Manifesto, see Sir Kristof Rodemai in the Town of Giran.
To get the Seal of Aspiration, see Sir Gustav Athebaldt in the Town of Oren

GM commands page:123

TELEPORT AND CIE

//tele - Opens the teleport menu.
//tele_areas - Opens the "special areas" teleport menu.
//show_teleport - Opens a basic teleport menu.
//goto name- Teleports you to the named player.
//recall name - Summons a named player to you.
//sendhome name - Sends a player to his HQ.
//recall_npc template - Teleport a NPC where you are.
//runmod tele - Teleport you instead of running.
//runmod norm - Come back to normal.
//ride [1-2] - Ride a mount (strider/wyvern).
//unride - Unride your current mount (as /dismount).
RELOAD TYPES / SERVER RESTART & SHUTDOWN

//server_shutdown time - Starts the shutdown process (in seconds).
//server_restart time - Starts the restart process (in seconds).
//server_abort - Stops the server shutdown / restart.
//unspawnall - Remove all npcs from world.
//respawnall - Reload npc data.
//reload dataType - Reloads given data type.
CHAOTIC WEAPONS

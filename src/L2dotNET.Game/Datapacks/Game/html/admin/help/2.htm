GM commands page:123

GENERAL ADMINISTRATION

//admin [1-4] - Brings up the admin menu.
//gmshop - Opens the GM shop.
//setew value - Sets enchantment of target player currently equipped weapon to value.
//bk [name] - Show bookmarks / Bookmark your current position.
//openall - Opens all doors.
//closeall - Closes all doors.
//itemcreate - Opens the item creation menu.
//create_item itemID [qty] - Creates an item.
//create_set itemID - Creates a full set based on chest id.
//reward_all itemID [qty] - Rewards the whole server with items.
GM TOGGLES

//gmlist - Toggle gmlist on/off.
//trade - Enable/disable trade asks.
//invul - Turns invulnerability on/off.
//hide - Turns invisibility on/off.
//silence - Enable/disable PM mode
//gm - Turns on/off your GM status.
NPCs / MOBs ADMINISTRATION

//show_spawns - Opens spawn menu.
//spawn npcID - Spawns a mob on current location (will respawn after server restart).
//spawn_once npcID - Spawns a mob on current location. Won't respawn again.
//spawn_index 1-80 - Shows menu for monsters with respective level.
//delete - Remove a targeted mob. Mob will not respawn.
//kill - Used to kill a targeted mob/player. Mob will respawn.
//show_scripts npcID - Shows events affecting NPC.
//show_skilllist npcID - Shows the NPC's skills list.
//show_droplist npcID - Shows the NPC's drops list.
//show_shop - Shows a targeted vendor's shoplist.
//show_shoplist - Shows a targeted vendor's shoplist.
